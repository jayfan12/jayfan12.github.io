2025.7.26

Je vous rejoindrai plus tard.
Elle fit pousser des roses.
Il a rendu l’argent.
Tom ne visitera pas Boston. source

  1. Je vous rejoindrai plus tard.
    我稍后会和你们会合。
    📘 知识点：

    • “rejoindrai” → 动词 rejoindre 的简单将来时 (futur simple)，第一人称单数。
    • “vous” = “你们 / 您”。
    • “plus tard” = “稍后 / 过一会儿”。

  2. Elle fit pousser des roses.
    她让玫瑰长了出来。
    📘 知识点：

    • “fit” → 动词 faire 的简单过去时 (passé simple)，第三人称单数。
    • “faire pousser” = “使……生长 / 种植”。
    • “des roses” = “一些玫瑰”。

  3. Il a rendu l’argent.
    他把钱还了。
    📘 知识点：

    • “a rendu” → 动词 rendre 的复合过去时 (passé composé)。
    • “rendre qch” = “归还某物”。
    • “l’argent” = “钱”。

  4. Tom ne visitera pas Boston.
    汤姆不会去波士顿。
    📘 知识点：

    • “visitera” → 动词 visiter 的简单将来时 (futur simple)，第三人称单数。
    • 否定形式：ne … pas 围住动词 → “不会……”。
    • “Boston” 专有名词，城市名。

2025.7.28

Laquelle est notre auto?
Il est très séduisant .
Où se trouve mon livre.
Elle l’accompagna au Japon.source

  1. Laquelle est notre auto ?
    哪一辆是我们的车？
    📘 知识点：

    • “laquelle” → 疑问代词，阴性单数，表示“哪一个”。
    • “notre auto” = “我们的车”。
    • 常用于在多种选择中询问。

  2. Il est très séduisant.
    他很有魅力。
    📘 知识点：

    • “séduisant” = “有吸引力的、迷人的”。
    • “très” = “非常”。
    • 句子结构：主语 + être + 形容词。

  3. Où se trouve mon livre ?
    我的书在哪里？
    📘 知识点：

    • “où” = “哪里”。
    • “se trouver” = “位于 / 在……”。
    • “mon livre” = “我的书”。

  4. Elle l’accompagna au Japon.
    她陪他去了日本。
    📘 知识点：

    • “accompagna” → 动词 accompagner 的简单过去时 (passé simple)，第三人称单数。
    • “l’” = “他/她/它”。
    • “au Japon” = “去日本”。
